#How do launch?

1. Install http://nodejs.org/ .
2. Install npm bower
    ```
     npm install -g bower
    ```
3. Get dependency
    ```
      bower install
    ```
4. Install basisjs-tool
    ```
       npm install -g basisjs-tools
    ```
5. Install gulp and gulp plugins
    ```
       npm install gulp
       npm install gulp-autoprefixer
       npm install gulp-stylus --safe
    ```
6. Run basis server
    ```
        basis server
    ```
7. Run gulp
    ```
       gulp
    ```
8. Open [tpmobile](http://localhost:8001/)
